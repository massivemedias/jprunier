KEY FILES DESCRIPTIONS
======================

1. CONFIGURATION FILES
   - vite.config.js: Build and development server configuration
   - package.json: Project metadata and dependencies
   - index.html: HTML entry point with meta tags

2. REACT APPLICATION
   - src/App.jsx: Main app component with routing setup
   - src/main.jsx: React DOM initialization

3. COMPONENTS (Reusable)
   - Header: Navigation bar with logo and mobile menu
   - Footer: Footer with CTA, links, and contact info
   - Hero: Full-width section with title, subtitle, CTA
   - ServiceCard: Card display for services with hover animation
   - TestimonialCard: Client testimonial with rating stars
   - ScrollToTop: Utility for navigation scroll behavior

4. PAGES (Route Components)
   - Home: Landing page with services, partners, sectors
   - About: Company information, values, testimonials
   - Services: Detailed service descriptions, Crestron expertise
   - Contact: Contact form and office locations
   - News: News feed and newsletter signup

5. DATA LAYER (JSON Files)
   - content.json: Company info, service descriptions, sectors, partners
   - testimonials.json: Client reviews with names, roles, avatars
   - news.json: News articles with titles, excerpts, dates

6. STYLING
   - global.css: Design tokens (colors, fonts, spacing), base styles, utilities
   - Component CSS files: Scoped styles for each component

7. BUILD OUTPUT
   - dist/ directory: Production-ready optimized bundle
